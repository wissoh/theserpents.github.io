2- Web development bases and HTML syntax

    Web development is the process of creating websites and connecting them to servers. Web development is defined under two terms, Backend (server work, code, database connection ...) and Frontend (what the user sees, UX, designs...). In this chapter we are gonna go through the backend stuff(code) especially the code that makes websites all over planet earth work HTML.

    Web development is like building a house on the internet! Just like a house has different parts, web development has different components too. The main parts of web development are HTML, CSS, and JavaScript. HTML is like the structure of a house, it defines the different sections and elements of a web page. CSS is like the paint and decorations, it adds colors, fonts, and styles to make the web page look beautiful. JavaScript is like the electricity that makes things come alive, it adds interactivity and allows you to create dynamic effects. When you visit a website, your web browser takes the HTML, CSS, and JavaScript code and turns it into a visual web page. Here's a simple diagram to help you understand:"

3- Data bases

    What is a Database?
    Imagine you have a big collection of information, like a library with many books. A database is like a digital version of that library. It's a place where you can store and organize your information in a structured way. In a database, you can store data such as names, numbers, text, and more.
    Python and Databases
    Python is a powerful programming language that can help you work with databases. Python provides libraries and modules that make it easy to connect to databases, create tables, insert data, retrieve data, and perform other operations.
    For example, the sqlite3 module in Python allows you to work with SQLite databases. SQLite is a simple database system that doesn't require a separate server. Here's a basic example of how to use Python to work with a database: 

    
import sqlite3

# Connect to the database (or create a new one if it doesn't exist)
connection = sqlite3.connect("mydatabase.db")

# Create a cursor object to execute SQL commands
cursor = connection.cursor()

# Create a table
cursor.execute("CREATE TABLE students (name TEXT, age INTEGER)")

# Insert data into the table
cursor.execute("INSERT INTO students VALUES ('Alice', 10)")
cursor.execute("INSERT INTO students VALUES ('Bob', 12)")

# Retrieve data from the table
cursor.execute("SELECT * FROM students")
results = cursor.fetchall()
for row in results:
    print(row)

# Close the connection
connection.close()

    Solution:

        import time

        my_time = int(input("Enter the time in seconds: "))

        for x in range(my_time, 0, -1):
            seconds = x % 60
            minutes = int(x / 60) % 60
            hours = int(x / 3600)
            print("{}:{}:{}").format(hours, minutes, seconds)
            time.sleep(1)
